Highland Line Split

npm install --save highland-line-split

Introduction

Splits a stream of strings into single lines. Watch:

_([
  "Test1\nTest2",
  "Test3"
]).consume(lineSplit()).toArray(function (res) {
  // res == ["Test1", "Test2Test3"]
})
